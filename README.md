# portal-aiyouxi-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面资源。这些页面以静态形式存储，不依赖后端服务，可直接在浏览器中打开使用。仓库内容主要面向网页资源存档与分发，不针对特定网站或域名。

## 目录说明

```
portal-aiyouxi-resource/
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面所需的样式、脚本、图片等静态资源
├── docs/           # 额外的文档或说明文件（可选）
└── README.md       # 本文件
```

- `pages/`：每个独立 HTML 页面以子目录或单独文件形式存放，便于管理。
- `assets/`：按需组织，与对应页面配套使用。

## 页面归档说明

- 所有页面均为纯静态 HTML，可直接通过 `file://` 协议或任意静态服务器访问。
- 页面内容可能包含外部资源引用（如 CDN 库），请确保网络环境可正常加载。
- 归档页面仅供学习、参考或存档用途，不保证实时更新或功能完整性。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/portal-aiyouxi-resource.git
   ```
2. 进入 `pages/` 目录，选择需要的 HTML 文件，双击或通过浏览器打开。
3. 也可将整个目录部署至任意静态托管平台（如 GitHub Pages）进行在线访问。
